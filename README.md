# Sport in Data Analysis

## Executive Summary
This project focuses on applying **data analysis techniques in sports**
to extract insights that support **performance evaluation and decision-making**.

---

## Project Objectives
- Understand how data analysis is applied in the sports industry
- Analyze sports-related datasets to identify performance patterns
- Evaluate player and team performance using analytical metrics
- Translate data findings into clear visual insights

---

## Dataset Overview
- Historical sports performance data
- Player and team statistics
- Match-level and season-level metrics

---

### 1. Data Collection
